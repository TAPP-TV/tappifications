Tappifications



Usage:

            
$.tappification({
    type: 'info',
    message: 'Your message goes here!',
    callToActionText: 'Click on me',
    callToActionUrl: 'http://www.tapptv.com/'
});

Requires

  • jQuery
  • font-awesome

Including in your project

CSS
            
css/tappifications.css
JS
            
js/tappifications.js
